# desafio-eteg

## Para rodar o desafio utiliz:

### docker compose up

## Entre no container da api e inicie a api

### docker exec desafio-eteg-api-1 bash

### npm run dev

## Entre no container da app e inicie o app

### docker exec desafio-eteg-app-1 bash

### npm run dev

## Para rodar os testes

### npm run test

## Documentação
